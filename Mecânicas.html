Mecânicas




Teclado


→ = Andar para a direita;

← = Andar para a esquerda;

↓ = Abaixar;

↑ = pular;

Q = aumentar velocidade;

W = Abrir o arsenal;

E = Atacar;

R = Golpe especial;

←← ou →→ = Dash para o lado;

↑↑ = Super pulo;

↑←← ou ↑→→ = Dash para o lado no ar;

E + E = Ataque duplo;

Acima de 3 “E” = Combo de ataques;

R + R = Ultra-golpe;

Pressionar o “R” = Transformação suprema do personagem.
(Só pode ser ativado caso tenha a barra de energia cheia)
